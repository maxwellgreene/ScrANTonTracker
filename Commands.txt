======================================
==========  DEWARP MANUAL  ===========
======================================
Put video in ScrANTonTracker/Video

Save calibration data on LAB, so that it can be used to dewarp other videos
---------------------------------
General:    python <py file path> "saveCalib" <projectName> <videoName> <dataFile = calibration_data>
FilePath:   /home/simulation/Documents/Github/ScrANTonTracker/ScrANTonTrackerLAB/Tracking/DewarpInterface.py
Example:    python /home/simulation/Documents/Github/ScrANTonTracker/ScrANTonTrackerLAB/Tracking/DewarpInterface.py saveCalib sample sampleVid1

Run Dewarp on LAB from calibration data
---------------------------------
General:    python <py file path> "dewarpVideo" <projectName> <videoName> <dataFile = calibration_data>
FilePath:   /home/simulation/Documents/Github/ScrANTonTracker/ScrANTonTrackerLAB/Tracking/DewarpInterface.py
Example:    python /home/simulation/Documents/Github/ScrANTonTracker/ScrANTonTrackerLAB/Tracking/DewarpInterface.py saveCalib sample sampleVid1
